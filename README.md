An AML / fraud detection project built step by step using the dataset `HI-Small_Trans.csv`(IBM HuggingFace).

## Current progress
- Project setup
- Data validation notebook
- Raw schema inspection
- Canonical renaming
- Missingness and target distribution checks
- Stable transaction ID generation

## Next step
- Chronological train / validation / test split

## Structure
- `notebooks/` → python notebooks
- `src/` → reusable Python code later
- `data/raw/` → raw dataset (not tracked in git)
- `data/interim/` → validated intermediate outputs (not tracked in git)
